# Triangle

## Description
The function aims to find the minimum path sum from the top to the bottom of a triangle. Each step can move to adjacent numbers on the row below.

```text
[
     [2],
    [3, 4],
   [6, 5, 7],
  [4, 1, 8, 3]
]
```

## Usage

Install pytest.
```bash
poetry install
```

Run the test.
```bash
poetry run pytest
```
